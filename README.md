# golf-framework

Golf coaching meta-router — 8 evidence-based frameworks covering Strokes Gained (Broadie), Course Management (DECADE), Mental Game (Rotella), Short Game (Pelz), Practice Design, Club Fitting, Physical Conditioning (TPI), and Scoring Strategy. No swing mechanics.

## Invoke

```
/golf
```

## Frameworks

| Skill | Focus |
|-------|-------|
| `strokes-gained` | Performance analytics (Broadie SG model) |
| `course-management` | DECADE decision framework |
| `mental-game` | Pre-shot routine, pressure management (Rotella) |
| `short-game` | Chipping, pitching, bunker (Pelz research) |
| `practice-design` | Deliberate practice and skill transfer |
| `club-fitting` | Static and dynamic fitting protocol |
| `physical-conditioning` | TPI movement screening and fitness |
| `scoring-strategy` | Risk-reward, par management, round planning |

## Requirements

- Claude Code with Skill tool enabled
- Installed via [claude-frameworks-marketplace](https://github.com/ironyjk/claude-frameworks-marketplace)
