# golf-framework

골프 코칭 메타 라우터 — 스트로크게인드(Broadie), 코스 매니지먼트(DECADE), 멘탈 게임(Rotella), 쇼트게임(Pelz), 연습 설계, 클럽 피팅, 피지컬 컨디셔닝(TPI), 스코어링 전략 8개 프레임워크. 스윙 메카닉 제외.

## 호출

```
/golf
```

## 프레임워크

| 스킬 | 내용 |
|------|------|
| `strokes-gained` | 퍼포먼스 분석 (Broadie SG 모델) |
| `course-management` | DECADE 의사결정 프레임워크 |
| `mental-game` | 프리샷 루틴, 압박 관리 (Rotella) |
| `short-game` | 칩핑, 피칭, 벙커 (Pelz 연구) |
| `practice-design` | 의도적 연습과 기술 전이 |
| `club-fitting` | 정적·동적 피팅 프로토콜 |
| `physical-conditioning` | TPI 움직임 스크리닝·피트니스 |
| `scoring-strategy` | 리스크-리워드, 파 관리, 라운드 플래닝 |
